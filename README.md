This is a Network Traffic Analysis dashboard that displays packet information from a Wireshark capture file that I made for my portfolio
*DO NOT USE ON PUBLIC NETWORKS IT MAY BE ILLEGAL*

To recreate:
  1. Create a .pcapng file on Wireshark and rename it traffic.pcapng. (If file is a .pcap file update the file name in parse.py line 6)
  2. In the terminal call parse.py
  3. In the terminal call plots.py -> this will create an html file which can be opened and viewed on a browser
